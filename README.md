和平精英游戏辅助，近距离的敌人方向提示，非透视，非挂，不用root，不修改数据，永不被封，只支持安卓8以上。

功能说明：
1.只有近处出现敌人的时候才会显示白色圆圈和红点

2.红点在白圈的上下左右位置，对应的敌人位置就是自身的前后左右方向
